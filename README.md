# practice_kaigo（架空LP）

介護施設の架空LP（ポートフォリオ作品）です。  
運用・保守の実務で意識している「情報の整理」「更新しやすさ」「確認しやすさ」を反映して制作しました。

- 公開URL：https://isogai0320wk-sys.github.io/practice_kaigo/

## 想定
- 初めて施設を探すご家族が、短時間で不安を解消し、見学予約まで迷わず進めること

## 実装内容
- セマンティックHTML（header / main / section / footer）
- レスポンシブ対応（900px以下でレイアウト変更）
- FAQのアコーディオン（jQuery）
- 画像差し替え（hero / map）

## 工夫した点（運用・保守目線）
- セクションを分割し、後から文言や順番を変更しやすい構成にした
- FAQは `aria-expanded` と `hidden` を使い、開閉状態が分かるようにした
- CSSは役割ごとにまとまるように記述し、影響範囲が追いやすいようにした

## 今後の改善
- ナビゲーションのモバイル対応（ハンバーガーメニュー）
- お知らせのカテゴリ絞り込み（JS）
- slick sliderの導入
